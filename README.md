# Maze Game

Простой 2D-лабиринт на Pygame. Управляйте зелёным квадратом-игроком с помощью стрелок. Стены обозначены символом `X`, выход — клетка `E` (красная). При столкновении со стеной происходит возврат на стартовую позицию.

## Демонстрация
- Окно 800x600
- Размер клетки/игрока: 20 пикселей
- Частота кадров: 60 FPS

## Установка и запуск

Требуется Python 3.9+ (рекомендуется) и pip.

```bash
# Клонировать репозиторий
git clone <repo-url>
cd MazeGame

# Создать и активировать виртуальное окружение (опционально)
python -m venv .venv
# Windows PowerShell
. .venv\\Scripts\\Activate.ps1

# Установить зависимости
pip install -r requirements.txt

# Запуск
python Maze_game.py
```

## Управление
- Стрелки: перемещение игрока
- Закрытие окна: выход из игры
